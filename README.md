# VERDICTO – Fake Review Detection System

A machine learning-based web application that detects fake (AI-generated) and real reviews from e-commerce platforms.

## Features
- Web scraping of product reviews using BeautifulSoup
- NLP preprocessing (tokenisation, stemming, lemmatisation)
- Feature extraction using Word2Vec
- Classification using Support Vector Machine (SVM)
- Flask-based API for real-time review analysis

## Tech Stack
Python, Flask, NLP, Word2Vec, SVM, Pandas, BeautifulSoup, NLTK

## How to Run
1. Install dependencies
2. Run app.py
3. Provide a product URL for analysis
